# N-cleo-x-Stellar-
Proyecto desarrollado en colaboración con Núcleo y Stellar, enfocado en la creación de soluciones descentralizadas utilizando Rust y el Soroban SDK. Este repositorio forma parte del programa "De 0 a dApp con Rust + Stellar", impulsando el aprendizaje práctico y el desarrollo de aplicaciones Web3 sobre la red Stellar.
